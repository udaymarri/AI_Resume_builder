# AI Resume Builder

An AI-powered Resume Builder that helps users create professional, ATS-friendly resumes in minutes. The application leverages AI to generate high-quality resume content, provides customizable templates, and allows users to edit, preview, and export resumes effortlessly.

## 🌐 Live Demo

**Live Application:** https://resume-builder-two-drab.vercel.app/

---

## 🚀 Features

* 🤖 AI-powered resume content generation
* 📄 ATS-friendly resume templates
* ✏️ Real-time editing with live preview
* 📥 One-click PDF export
* 🔐 Secure user authentication
* ☁️ Cloud-based resume storage
* 📱 Fully responsive design

## 🎯 Usage

1. Sign in to the application.
2. Enter your personal, educational, and professional details.
3. Generate AI-assisted resume content.
4. Customize the resume with professional templates.
5. Preview the resume in real time.
6. Export the final resume as a PDF.
